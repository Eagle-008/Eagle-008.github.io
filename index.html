Windows начал блокировать uTorent, приняв программу за вирус

Windows начала распознавать в популярном торрент‐клиенте uTorrent вредоносную программу, передает телеканал "360". Все современные версии Windows содержат антивирусную программу Defender. По версии антивируса, торрент-клиент — "потенциально нежелательное программное обеспечение". Так же считают антивирусы NOD32, Sophos, TrendMicro и другие. По данным Microsoft, uTorrent негативно влияет на производительность вашей системы.
